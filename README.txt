		--English--
The Helper AI extension is powered by AI. It allows you to condense your texts while retaining key details and the main idea.
The text summarization is driven by the Qwen 2.5 14B model.

To install the extension, you need to enable Developer mode in the Extensions section, and then click the "Load unpacked" button (the extension must be located in a folder in a convenient place for you).

To activate it, you need to obtain an API key for this neural network:
First, open the HuggingFace website and go to the Qwen section (make sure you are logged in).Click on your profile avatar and select "Access Tokens" below.
Create a new token, choose "Fine-grained" as the type, and under the "Interface" permissions, check "Make calls to Inference Providers".
Paste the received code into the extension settings and save it (the key will persist after a restart).
To condense a text, highlight it, right-click, and select "Summarize with Helper AI". 
Then, open the extension—the text will already be processed or in the process of being handled by the AI.
		--Russian--
Расширение Helper AI построено с помощью ИИ.С помощью него вы можете сжимать ваши тексты оставляя подробности и главную мысль. Модель которая сжимает тексты-Qwen 2.5 14B.

Для установки в разделе расширения нужно включить режим разработчика,а затем нажать кнопку "Загрузить распакованное расширение"(Расширение должно находится в папке в удобном вам месте)

Для активации вам нужно получить API-ключ для данной нейросети:
1. Для начала открывает сайт HuggingFace в разделе Qwen (нужно быть залогиненым).
2. Жмем на аватарку профиля и ниже выбираем "Acess Tokens"
3. Создаем новый токен и типе выбираем "Fine-grained",а в разрешениях в разделе Interface "Make calls to Inference Providers".
4. Полученный код вставляем в настройки расширения и сохраняем его (при перезагрузке ключ не теряется).
5. Для сжатия текста нужно выделить его,нажать ПКМ и "Summarize with HElper AI",а затем открыв расширение текст уже будет обработан/обрабатываться нейросетью.
